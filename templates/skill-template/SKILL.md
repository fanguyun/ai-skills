# 技能名称

用 1 到 2 段说明这个 skill 的目标、适用范围和默认假设。

## 快速流程

1. 确认用户给的对象、范围和目标。
2. 先读取必要源码、页面、截图或上下文。
3. 只加载当前任务需要的 `references/`、`scripts/`、`assets/`。
4. 输出结果时优先给出可执行建议，而不是泛泛描述。

## 输出要求

- 按优先级组织结果
- 明确已确认问题与推断风险
- 尽量给出最小可行修复方案
